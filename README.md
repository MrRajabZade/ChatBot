# ChatBot
به چت بات آموزش دهید و چت بات خود را بسازید.

مثال برای rulse.txt:
```
iran = eeee
Irani = p
+
سلام : خوبی؟
iran? : #iran #Irani
my * is * : * = *
خوبی {فلانی,داداش} : آره
حالم بده : بیماری (بله & متاسفم (چه کار کنم؟ & برو بخواب (بخوابم؟ & آره (وای سرگیجه دارم & حالت خوبه؟))))
چطوری : خوبم [مرسی,تشکر]
```
